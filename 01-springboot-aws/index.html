Intellij가 이클립스보다 더 좋은 이유

  • 강력한 추천 기능 (Smart Completion)
  • 다양한 리팩토링과 디버깅 기능
  • 이클립스 Git에 비해 높은 자유도
  • 프로젝트 시작시, 인덱싱을 통해 파일을 비롯한 자원들에 대한 빠른 검색 속도
  • HTML, CSS, JS, XML에 대한 강력한 기능 지원
  • 자바, 스프링 부트 버전업에 맞춘 빠른 업데이트 등

좀 더 Intellij에 대해 알고 싶은 경우.

  • 젯브레인 한국 사용자 모임
  • 책 저자 블로그
  • 인프런 인텔리제이 가이드
