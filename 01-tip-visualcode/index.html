Visual Studio Code 터미널 실행

항상 초기 터미널 세팅을 할 때, 반드시 하는 세팅 중 하나인데 자주 까먹어서 작성합니다.

진행 순서

  1. Visual Studio Code 실행
  2. Command Pallete를 엽니다.
  • Mac : shift + command + P
  1. Shell Command 를 입력해서 [Shell Command: Install 'code' command in PATH] 를 실행합니다.
  2. 터미널에서 원하는 디렉토리에서 [code .]를 입력시 실행가능합니다.
